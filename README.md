# ななぷれ｜夏のボーナスで選びたい！ごほうびバッグ特集ページ

楽天GOLD（RMS）向けの特集ページ一式です。1画面ごとに送るフルスクリーン型（縦スナップ）のLPで、PC・スマホ両対応です。

---

├─ style.css         … デザイン（CSS）
├─ script.js         … 動き（JavaScript。jQuery等の外部ライブラリ不要）
├─ README.md         … この説明書
└─ images/           … 画像一式（16点）

## 2. 楽天GOLDへのアップロード方法

### パターンA：フリーページとして丸ごと使う（推奨・かんたん）
1. RMS →「楽天GOLD」→ FTPで下記をアップロード
   - `index.html` / `style.css` / `script.js`
   - `images/` フォルダ（中の画像すべて）
2. `index.html` と同じ階層に `style.css`・`script.js`・`images/` を置けば、そのまま表示されます。

### パターンB：既存GOLDテンプレートの「フリーHTML」枠に貼り付ける
- `index.html` 内の `<div class="scroll-container">` 〜 `</div>` 以降のみを貼り付け
- `style.css` / `script.js` は店舗の共通ファイルとして読み込んでください

> 画像パスは `images/○○.jpg` の相対パスです。GOLDの実際の画像URL（`https://www.rakuten.ne.jp/gold/nanaple/...` など）に置き換える場合は、HTML内の `images/` をまとめて置換してください。

## 3. よく使う差し替え箇所

HTML内には差し替え箇所に `▼差し替え▼` のコメントを付けています。検索して置き換えてください。

| 変更したいもの | 場所 |
|---|---|
| 表紙画像 | `index.html`：`images/kanban2.jpg` |
| 商品画像 | 各商品セクションの `product-image` の `src` |
| 価格 | `<span class="price-value">42,480</span>` の数字 |
| 商品ページURL | 「詳しく見る」ボタンの `href` |
| お気に入り登録 | `btn-favorite` の `href`（RMS発行のお気に入り登録コードに差し替え可） |
| 目次の見出し文 | `index.html` 末尾の `hero-toc-list` 内 |

### 商品を増やす／減らすとき
1. `index.html` の `<!-- ---------- 商品 N ---------- -->` のセクション（`<section ... id="product-N">`）を複製・削除
2. 複製したら `id`（`product-7` など重複しない値）・画像・価格・URLを差し替え
3. ページ末尾の目次（`hero-toc-list`）の項目も合わせて増減

### 色を変えたいとき
`style.css` 冒頭の `:root` にブランドカラー等をまとめています。
- `--color-accent`：アクセントカラー
- 黄色の背景：`#fcff0a`（`style.css` 内を検索して一括変更可）

## 4. 今回の主な調整点（PC表示・スマホ操作の改善）

- **PC目次パネル**：画面の高さに追従して常に全項目が収まるように。幅をスリム化し、ブラウザの表示倍率（90%/100%/125% 等）を変えてもレイアウトが崩れにくくしました。
- **最下部（SHOPトップ）の白背景**：上の商品写真と同じ幅・同じ位置にそろえ、表示倍率を変えても揃い続けるようにしました。
- **価格・お気に入りボタン**：PCで拡大表示（100%超）しても、価格・ボタンが商品写真に被らないよう、余白帯に収まるサイズへ自動調整。
- **スマホのスクロール**：軽いスワイプでも反応し、下に送りやすくしました。

---

## 5. 微調整用のパラメータ（必要な方向け）

コード内のコメントで調整できます。

| 調整したい内容 | ファイル | 目印 |
|---|---|---|
| 表紙のバッグが流れる速さ | `style.css` | `heroBagScroll ...s`（秒数を大きくすると遅く） |
| 目次パネルの幅 | `style.css` | `--toc-width` |
| 価格・ボタンの高さ位置（PC） | `script.js` | `GAP_RATIO_PC` |
| スマホのスワイプ感度 | `script.js` | `SWIPE_THRESHOLD` / `TOUCH_LOCK_DURATION` |
| セクションの送り速度 | `script.js` | `DURATION` |

## 7. 注意点

- お気に入りボタンは仮のリンクが入っている場合があります。公開前にRMSで発行される「お気に入り登録」ボタンのコードに差し替えてください。
- ブラウザにキャッシュが残っていると古い表示になることがあります。更新後は **Ctrl+F5（スーパーリロード）** で確認してください。
